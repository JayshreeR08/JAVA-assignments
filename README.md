# Java Programming Lab 

This repository contains all the Java programs assigned in my lab coursework.  
The programs cover core concepts of Java including multithreading, Swing (GUI), JDBC, exception handling, file handling, and OOP principles.

---

## Topics Covered

- Multithreading (Thread & Runnable)
- Thread Synchronization using join()
- Swing (GUI Applications)
- JDBC Connectivity (Database)
- Exception Handling
- User Defined Exceptions
- File Handling
- Packages and Sub-packages
- OOP Concepts (Inheritance, Interface, Abstract Classes)

---

## List of Programs

1. Multithreading using Thread class  
2. Multithreading using Runnable interface  
3. Thread synchronization using join()  
4. Addition of two numbers using Swing  
5. Registration Form with JDBC connectivity  
6. Calculator using Swing  
7. Matrix Addition using Swing  
8. Shapes creation using buttons  
9. Paint application using Mouse Events  
10. Package with 5 classes  
11. Package and Sub-package implementation  
12. Exception Handling (Array & Arithmetic)  
13. User Defined Exception (Age validation)  
14. File Handling programs  
15. Inheritance, Interface & Abstract Class implementation
